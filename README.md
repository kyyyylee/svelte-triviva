# Vite + Svelte Trivia app
Using Vite, Svelte and Open Trivia API!


## To run docker: 

```bash
git clone https://github.com/DWDD-3620-Spring-2024/svelte-trivia-app-kyyyylee
```

```bash
cd svelte-trivia-app-kyyyylee
```

```bash
npm install 
```

```bash
npm run dockerBuild
```

```bash
npm run dockerRun
```
